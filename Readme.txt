Telephone-Iphone


Composition du dossier :
- un fichier HTML5
- un dossier CSS comportant un fichier CSS3
- un dossier JS comportant deux fichiers JavaScript (script.js pour l'interactivité du téléphone - moment-with-locales.js pour l'heure du téléphone)
- un dossier image comportant 7 images

Deux API ont été utilisées afin d'obtenir les informations météorologiques ainsi que l'heure exacte.
Météo en temps réel : https://www.visualcrossing.com
Heure et date en temps réel : https://momentjs.com


C'est une reproduction d'un téléphone Iphone de la marque Apple.
Le téléphone Iphone a été créé uniquement avec les langages de programmation HTML5 et CSS3.

Le téléphone est interactif : il s'allume et s'éteint à l'aide du bouton situé en bas du téléphone.


Une fois allumé le téléphone affichera :

- les icônes habituelles d'un téléphone Iphone (wifi, batterie, message, appel, Safari, Itunes)
- l'heure exact en heure, minute et seconde
- une barre de recherche Google fonctionnelle
- un logo météo sur lequel il est possible d'appuyer afin d'obtenir les prévisions météorologiques de la ville de Mulhouse (France)

Si l'on éteint le téléphone, tous les paramètres d'affichages et positions des éléments se remettront automatiquement à zéro.
